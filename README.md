# ⚡ Electric - Browser

**Electric** is a powerful, easy-to-use web proxy built for students to unblock websites. Electric proxy is not fully done but you can still access the beta version

---

## 🌐 Features

- 🔓 Access blocked websites
- ⚡ Fast and lightweight Node.js backend
- 🎮 Games and apps + Now.gg support
- 🌙 Sleek, responsive UI with dark mode support
- 🔐 Privcy, using electric your ip and everything is hidden and secured.
- 

---

## 📦 Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Proxy Engine**: [Corrosion](https://github.com/titaniumnetwork-dev/corrosion) / [Ultraviolet](https://github.com/titaniumnetwork-dev/ultraviolet) *(customizable)*
